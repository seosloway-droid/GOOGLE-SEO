# SEO Analysis Files

This folder contains SEO NLP analysis reports generated by the SEO NLP Analyzer app.

## How to use with Claude Code

When user says "check analizo" or "priporocaj kako naprej", do the following:

1. Find the most recent `.md` file in this folder
2. Read the full analysis report
3. Based on the data, provide:
   - **Top 3 priority actions** ranked by SEO impact
   - **Specific rewrite suggestions** for the biggest problems found
   - **Content gap analysis** — what topics/entities are missing
   - **Target values** — what numbers to aim for (sentiment, lexical density, etc.)
   - **Quick wins** — what can be fixed in under 1 hour

## File naming convention

Files are named: `analiza_{keyword}_{timestamp}.md`

Example: `analiza_bazeni_20260509_143022.md`

## Legend in reports

- 🔵 Official Google data — directly from Google NLP API
- 🟠 SEO best practice — industry standard, not officially confirmed by Google  
- 🟣 Google guidelines — from Google's Quality Rater Guidelines
